# 📊 Análise de Dados de Marketing Bancário (MySQL)

## 📌 Problema de Negócio
Instituições financeiras realizam campanhas de marketing para promover produtos, porém nem todos os clientes aderem às ofertas.  

Este projeto tem como objetivo analisar dados de clientes utilizando SQL para identificar padrões que influenciam a adesão a depósitos a prazo.

---

## 🎯 Objetivos
- Entender o comportamento dos clientes em campanhas de marketing  
- Identificar fatores que influenciam a conversão  
- Analisar características demográficas e financeiras  
- Apoiar decisões estratégicas baseadas em dados  

---

## 📂 Base de Dados
O conjunto de dados contém informações sobre clientes e campanhas, incluindo:

- Dados demográficos (idade, profissão, estado civil, escolaridade)  
- Informações financeiras (empréstimos, crédito)  
- Dados da campanha (tipo de contato, mês, duração, histórico)  
- Variável alvo: adesão ao depósito (sim/não)  

---

## 🛠️ Ferramentas Utilizadas
- MySQL  
- SQL (JOIN, GROUP BY, agregações, subqueries)  

---

## 🧹 Exploração e Tratamento de Dados
- Análise da estrutura dos dados  
- Identificação de variáveis categóricas e numéricas  
- Tratamento de valores “unknown”  
- Filtragem de dados relevantes para análise  

---

## 📊 Principais Análises

### 📌 Perfil dos Clientes
- Distribuição por profissão, estado civil e escolaridade  
- Identificação de perfis com maior taxa de conversão  

### 📌 Performance das Campanhas
- Comparação entre tipos de contato (telefone vs celular)  
- Identificação dos melhores períodos para campanha  

### 📌 Comportamento Financeiro
- Impacto de empréstimos e crédito na decisão do cliente  
- Relação entre perfil financeiro e conversão  

### 📌 Influência de Contatos Anteriores
- Clientes já contatados anteriormente apresentam maior chance de conversão  

---

## 📈 Principais Insights

- Clientes mais velhos tendem a apresentar maior taxa de adesão  
- Histórico positivo em campanhas anteriores aumenta significativamente a conversão  
- O canal de contato influencia diretamente os resultados  
- Determinados perfis de clientes apresentam maior engajamento  

---

## 🧠 Recomendações de Negócio

- Direcionar campanhas para perfis com maior probabilidade de conversão  
- Priorizar canais de contato mais eficientes  
- Utilizar dados históricos para melhorar a segmentação  
- Otimizar o timing das campanhas  

---

## 🚀 Possíveis Melhorias

- Construção de modelo preditivo de conversão  
- Integração com ferramentas de BI (Power BI)  
- Automatização de relatórios com SQL
